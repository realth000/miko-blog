# Sample about page

Edit and save your own about page content in `about.mdx`.

## Introduction

This is a blog built with [miko-blog](https://github.com/realth000/miko-blog).

miko-blog is:

- a toy static site generator.
- for generating static blogs from `mdx` files.
- based on `react`, `typescript`, `mdx.js`, with as less dependencies as possible.

For `mdx` contents:

- Each `mdx` file serves as an article.
- Each article has a series of tags describe its content.
- miko-blog also provides projects show page for your work.

## Usage

Follow these steps to use miko-blog:

1. Configure the blog info in `src/values/config-values.ts`.
   - See `src/values/config-values.example.ts` for example.
2. Prepare blog articles. Each article should be named `index.mdx` in a subdir in `src/contents`, together with a json config file in the same directory.
   - Directory structure for a article.
     ```console
     src/contents
     └── sample
         ├── config.json
         └── index.mdx
     ```
   - Example `config.json`:
     ```json
     {
       "title": "Sample",
       "summary": "Sample document",
       "date": "2026-01-01",
       "tags": ["mdx", "sample blog"],
       "draft": false
     }
     ```
   - The directory name can be anything not only the article name, e.g. name the folder `001-sample` for sorted order.
3. Run `pnpm gen-doc` to run code generation.
4. Run `pnpm dev` to build and serve your blog.

/// I am symbol level doc comment.
#[derive(Debug)]
struct StructHere<'a> {
    /// Doc comment on field.
    ref_str: &'a str,
}

pub trait MyTrait {
    type Out;

    fn use_the_type(i: u32);
    fn use_it(&self) -> Option<bool>;

    fn default_implemented_one(&self) -> Self::Out {
        unimplemented!()
    }
}

impl<'a> MyTrait for StructHere<'a> {
    type Out = &'a str;

    fn use_the_type(i: u32) {
        println!("use the type: {i}")
    }

    fn use_it(&self) -> Option<u32> {
        if self.ref_str.starts_with("foo") {
            Some(100)
        } else {
            None
        }
    }
}

/*
 * Normal multiline comment
 */
pub fn run_the_func() -> String {
    let sh = StructHere::new("here is the ref str");
    match sh.use_it() {
        Some(v) if v >= 10 => println!("use_it true"),
        Some(_) => println!("use_it value"),
        None => println!("use_it none"),
        _ => println!("use_it dont care"),
    }

    let i = 2 / 1;

    // Plain comment.

    my_macro!(100);

    sh.ref_str
        .chars()
        .filter(|ch| ch.is_alphabetic())
        .map(|ch| ch.to_string())
        .collect::<Vec<_>>()
        .join(",")
}
